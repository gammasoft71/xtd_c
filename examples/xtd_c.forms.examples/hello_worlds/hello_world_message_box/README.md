# hello_world_message_box

The classic first application "Hello, World!" with  [xtd_forms_message_box](../../../../src/xtd_c.forms/include/xtd/forms/message_box.h) control.

# Sources

* [src/hello_world_message_box.c](src/hello_world_message_box.c)
* [CMakeLists.txt](CMakeLists.txt)

# Build and run

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```shell
xtdc run
```
